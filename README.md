[RED ME]{.underline} ABOUT **ShopSphere API**

ShopSphere API is a complete E-Commerce backend system built using
ASP.NET Core Web API and Entity Framework Core.

Tech Stack

-   ASP.NET Core Web API

-   Entity Framework Core

-   SQL Server

-   JWT Bearer Authentication

-   LINQ

-   Swagger / Scalar

**[Technologies Used]{.underline}**

-   ASP.NET Core Web API (.NET)

-   Entity Framework Core (ORM)

-   SQL Server (Database)

-   JWT Authentication (Security)

-   LINQ (Data Querying)

-   RESTful API Architecture

-   Swagger / Scalar (API Documentation)

-   Clean Architecture (Controllers, Services, Repositories, DTOs)

**[Project structure]{.underline}**\
Controllers\
Models\
DTOs\
Services\
Repositories\
Data\
Program.cs

**[API End Points]{.underline}**

### Auth

-   POST /api/auth/register

-   POST /api/auth/login

### Products

-   GET /api/products

-   POST /api/products

-   GET /api/products/{id}

-   PUT /api/products/{id}

-   DEL /api/products/{id}

### Orders

-   GET /api/orders

-   POST /api/orders
